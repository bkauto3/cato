# Cato audit and verdict documents

This directory is the index for audit pipeline outputs and Kraken/Alex verdicts.

## Pipeline

- **Audit pipeline (how to run)**: [../audit-pipeline.md](../audit-pipeline.md)
- **Alex** produces `CATO_ALEX_AUDIT.md` at repo root.
- **Kraken** produces `CATO_KRAKEN_VERDICT.md` at repo root (and optionally `KRAKEN_VERDICT_*.md` for historical tracks).
