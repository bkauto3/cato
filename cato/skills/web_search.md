# Web Search
**Version:** 1.0.0
**Capabilities:** browser.search, browser.navigate

## Instructions
Search the web using DuckDuckGo and return relevant results.

When asked to search for something:
1. Use the `browser` tool with action `search` and the query
2. For each result, optionally navigate to the top 1-2 results for more detail
3. Summarize findings in a clear, structured response
4. Always cite the source URLs

Never use Brave Search or Google Search APIs.
